## Configuration

### 1. Récupérez la clé API

#### 1.1 Accéder à l'espace développeur

1. Connectez-vous à votre compte PolluScan  
2. Dans le menu principal, rendez-vous sur la page  /developer

#### 1.2 Cliquez sur Créer une clé API

![Get Your API](../images/api-key.png)

Nommez la clé API générée et assurez-vous de la sauvegarder.
